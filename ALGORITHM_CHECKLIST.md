# Algorithm Implementation Checklist

This checklist tracks baseline algorithms and data structures to implement in this repository.
The goal is to collect reusable fundamentals, not LeetCode- or Codeforces-specific problem solutions.

## How to Use This Checklist
- Pick the next unchecked item from the recommended sequence unless there is a specific reason to change priority.
- Implement each item as a standalone learning topic with the repository's standard structure: implementations, inputs, `PROBLEM.md`, `USAGE.md`, and Makefile targets.
- Treat referenced problem families only as motivation for why the algorithm matters.
- Prefer baseline, canonical implementations before advanced optimizations or hybrid problem-specific variants.
